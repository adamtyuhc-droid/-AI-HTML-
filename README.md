# 📚 Learning Notes — Claude Code Plugin

將任何教材內容，轉成精美的繁體中文互動 HTML 學習筆記。

---

## 安裝方式

```
/plugin install path/to/stats-notes-plugin
```

或解壓縮後安裝：

```
/plugin install path/to/stats-notes-plugin-v1.zip
```

---

## 第一次使用

安裝後，直接說「存」或「幫我存」，Claude 會問你：

> 「請問你想把學習筆記存在哪個資料夾？」

輸入一次路徑（例如 `C:\Users\你的名字\Desktop\學習筆記`）後，系統會：
1. 記住這個路徑
2. 自動建立 `notes.html` 筆記主檔
3. 之後不再詢問，直接寫入

---

## 能做什麼

### ✦ 把教材變成互動筆記
- 傳圖片、截圖、打字給 Claude，說「存」就會自動整理成筆記頁面
- 支援任何科目：數學、統計、物理、化工、程控、語言…都可以

### ✦ 自動建立科目分類
- 第一次出現新科目，自動在 sidebar 新增分類（不需要手動設定）
- 每個科目有自己的顏色圓點，一眼區分

### ✦ 精美的深色主題 HTML
- **KaTeX 數學渲染**：`$E=mc^2$`、`$$\int f\,dx$$` 自動顯示為漂亮公式
- **Sidebar 三層導覽**：科目 → 章節 → 頁面
- **首頁卡片總覽**：所有科目一目了然
- 完全離線可用（只需要瀏覽器）

### ✦ 結構化的筆記格式
每個頁面自動包含：
| 區塊 | 顏色標籤 | 說明 |
|------|---------|------|
| 定義 | 🔵 藍 | 術語定義、符號說明 |
| 定理 | 🟣 紫 | 公式、結論 |
| 推導 | 🟡 琥珀 | 步驟推導、方法 |
| 例題 | 🟢 綠 | 數值計算、例子 |
| 直觀解釋 | 🔴 珊瑚 | 白話說明為什麼這樣 |

### ✦ 增量插入，不會覆蓋舊筆記
- 每次「存」只插入新內容，原有筆記完整保留
- 用 HTML 註解標記精確定位插入位置

## 使用範例

**場景 1：上課做筆記**
```
（拍一張講義照片）
→ 說「幫我存」
→ Claude 整理成筆記頁面，存入 notes.html
```

**場景 2：看完一個單元**
```
「教我完再存，這是 Ch.3 的線性化」
→ Claude 教完後，自動整理並存入 Ch.3 對應位置
```

**場景 3：新科目**
```
（上傳統計學講義）
→ 說「這是統計學，幫我存」
→ 自動建立「統計學」科目區塊，存入第一頁
```

---

## 筆記檔案位置

安裝後所有筆記存在你指定的資料夾，例如：
```
C:\Users\你的名字\Desktop\學習筆記\
└── notes.html    ← 用瀏覽器開啟即可
```

## 配合使用的 Skill

| Skill | 功能 |
|-------|------|
| `stats-notes`（本 plugin）| 理論推導、公式、定義、例題 |

## 技術細節

- 純 HTML + CSS + JavaScript，無需後端
- 字型：Fraunces（標題）、DM Mono（程式碼/標籤）、Noto Serif TC（中文內文）
- 數學：KaTeX 0.16（CDN，需網路首次載入）
- 相容所有現代瀏覽器
